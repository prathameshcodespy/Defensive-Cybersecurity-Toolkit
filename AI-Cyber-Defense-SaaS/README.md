# AI Cyber Defense SaaS Starter Kit

Production-ready cybersecurity SaaS platform built with Flask.

## Features
- JWT Authentication (Access + Refresh Tokens)
- Role-based Admin Panel
- Email Automation
- Live Cyber UI Dashboard
- AI Risk Simulation
- Deployment Ready

## Installation

1. Clone project
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python app.py

## Admin Access
Create user with role "admin" manually in database.
